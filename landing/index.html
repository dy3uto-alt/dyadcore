什么是 dyadcore？

dyadcore 是一个为 AI Agent 设计的本地 Peer 记忆系统。它不依赖任何外部服务——纯 SQLite 驱动的语义检索，单文件即用。

对话中，用户和 Agent 如同磁场的两极，各自留下痕迹。这些痕迹之间形成关系网——triggered、echoed、contradicted、related——构成了一个动态演化的认知图。

Dual Mirror 架构

🪞

User Mirror

source="user"

记录用户的行为、偏好、表达——每一次交互都是一条痕迹。

🪞

Agent Mirror

source="agent"

记录 Agent 的观察、推理、自我反思——对等实体，无特殊类别。

Relation Network

跨轨迹的关系边构成了记忆之间的语义网络。每条边都是一个认知信号。

triggered echoed contradicted related

Quick Start

from dyadcore import DyadCore

dc = DyadCore("agent_memory.db")

# 写入痕迹
uid = dc.write("用户偏好本地部署", source="user", field="技术选型")
dc.anchor(uid)  # 锚定重要记忆

# 语义检索 + 图扩展
results = dc.recall("本地记忆方案", field_hint="技术选型")

dc.close()

核心特性

01

FTS5 Trigram 检索

SQLite 内置全文索引，零成本，中文友好。短词 LIKE 回退兜底。

02

关系图扩展

检索时自动沿 reflections 边做 1-hop 语义邻居发现，召回相关但不直接匹配的记忆。

03

Beliefs 认知层

从关系图拓扑中涌现的信念系统——自动合成、可确认、可挑战、可取代。

04

零依赖

无 sqlite-vec、无 embedding 模型、无 C 扩展、无后台服务。仅需 Python 标准库。

05

批量写入

write_batch() 单事务批量导入，5-10x 提速。适用于历史对话回填。

06

Field Snapshot

关系场动态分析——场强、极性、重心、关系密度。minimal 的可观测性。
